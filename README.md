# Descripción del proyecto.
Esta aplicacion determina según el porcentaje de uso de un nombre, si el mismo se aplica a masculino o femenino.

# Exposición del problema
Este desarrollo está destinado a solucionar un problema frecuente cuando se manejan clientes de diferentes partes del mundo. 
El mismo refiere a que es frecuente no saber como nominar a una persona, si Sr. o Sra. por ejemplo. 
Con esta app se logra aplicar el género de forma correcta.

# Plataforma y Tecnología
El desarrollo se ha llevado a cabo en Kotlin y se enfoca en Android. 

# Interfaz de usuario
Con una interfaz bien simple y super práctica esta aplicación solicita un nombre y devuelve el género del mismo.

# Funcionalidad
La aplicación presenta una pantalla sencilla y muy amigable donde es posible ingresar un nombre para analizar. 
El nombre ingresado es analizado a través de una API externa a un sitio especializado que tiene el acumulado mundial de nombres y generos utilizados para esos nombres.
El resultado es devuelto en un campo que identifica el género y lo califica como masculino, femenino o indeterminado si el nombre no es ubicado.

# Instalación
La aplicación está disponible para ser instalada directamente en un dispositivo Android.

# Conexiones con API externas
La operativa de consulta utiliza una API de genderize.io
